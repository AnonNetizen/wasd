# /update-memory

## 用途

按规则 14-B / ADR #15 / ADR #19，AI 应主动更新 `docs/AI记忆/项目记忆.md` 与会话日志。本命令是显式兜底。

## 步骤

1. 读 `docs/AI记忆/项目记忆.md`、`docs/决策记录.md`、`docs/游戏设计文档.md`、`docs/CICD规划.md` 与最近 git commit log。
2. 对照版本号、ADR 条数、工具链、下一步候选、近期对话脉络是否过期。
3. 修订项目记忆：同位置覆盖而非追加；主文件保持 ≤ 200 行。
4. 写当日 `docs/AI记忆/会话日志/<YYYY-MM-DD>.md`。
5. 只记结论 + 关键参数 + 文件指针，不复述完整对话。

## 不要做

- 不要自动 commit。
- 不要改 ADR / 规则 / 设计文档，除非这是当前主任务的一部分。
- 不要把历史细节塞进项目记忆主索引。
