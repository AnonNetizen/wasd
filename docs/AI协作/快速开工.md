# 快速开工（低 token 热路径）

> **AI 修改说明**：修改本文档前先读 `docs/AI协作/文档维护指南.md`。
> 本文档是 AI 日常接手项目的低 token 热路径；改开工步骤、热 / 冷分层或任务路由时，必须同步 `AGENTS.md`、平台入口、`docs/AI导航.md`、`docs/AI协作/README.md`、`docs/AI协作/上下文预算.md`、`docs/AI记忆/current_state.json`。

---

## 1. 日常读法

日常新会话先读：

1. `AGENTS.md` 或当前平台入口（`CLAUDE.md` / `CODEX.md` / `OPENCODE.md`）
2. 本文件
3. `docs/AI记忆/current_state.json`
4. 当前平台编码规则入口
5. `docs/AI导航.md` 中与任务相关的段落

完整长期文档按任务触发再读，不默认整篇加载。

## 2. 不可跳过

- 用户最后明确指令优先于摘要、`Next Steps` 和 `current_state.json` 候选项。
- 需求、术语、验收标准、授权边界或上下文含义不清时，先问一个简短澄清问题；不要为了推进任务而自行脑补高风险假设。
- `draft/` / `DRAFT/` 是人工草稿禁区，除非用户明确授权。
- 数据驱动、本地化、InputMap、词表常量、autoload 边界、文档同步、测试义务与 AI Git 策略不变。
- 当前 shell 为 PowerShell 时，执行命令前遵守平台规则第 29 节；固定字符串优先 `rg -F`，预期非零退出码先归一化，模板见 `docs/AI协作/工具适配指南.md`。
- 改规则、入口、ADR、GDD、测试策略、词表或长期文档前，先读对应权威全文和 `docs/AI协作/文档维护指南.md`。

## 4. 按任务补读

| 任务 | 额外读取 |
|------|----------|
| 续接当前任务 | `current_state.json`、当日会话日志，需要长期背景时再读 `项目记忆.md` 相关节 |
| 推进正式项目 | 优先读当前阶段工作包；当前是 `docs/AI协作/工作包/F13-HandcraftedRooms.md` |
| 改代码模块 | `docs/代码文档规范.md`、对应 `docs/代码/<module_id>.md`、目标源码；GDD / ADR 只在设计冲突或新增决策时补读 |
| 改数据 / 文案 | `client/data/README.md` 或 `client/locale/README.md`、目标数据文件、`docs/词表与契约.md` 相关节 |
| 改设计 / 决策 / 规则 | GDD / ADR / 平台规则等对应权威全文、`docs/AI协作/文档维护指南.md` |
| 高频任务 | `docs/AI协作/任务模板/<任务>.md` |
| 非高频任务 | `docs/AI协作/上下文预算.md` |

## 5. 冷存储

- 长期事实索引：`docs/AI记忆/项目记忆.md`
- 每日细节：`docs/AI记忆/会话日志/YYYY-MM-DD.md`
- 权威层级与任务路由：`docs/AI知识库索引.md` / `docs/_kb_index.json`
- 历史验证细节：会话日志；`current_state.json.last_verified` 只保留最近热路径验证。

## 6. 常用验证

- JSON：`python -m json.tool docs/AI记忆/current_state.json`
- 知识库索引：`python -m json.tool docs/_kb_index.json`
- 文档健康：`python tools/docs_health_check.py`
- 契约同步：`python tools/sync_contracts.py --check`
- 数据校验：`python tools/validate_data.py`
- Godot 启动：`python tools/godot_bridge.py --project client headless-boot`
- 空白检查：`git diff --check -- . ":(exclude)draft/**" ":(exclude)DRAFT/**"`
