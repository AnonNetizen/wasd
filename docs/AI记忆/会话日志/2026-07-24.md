# 2026-07-24 会话日志

## ADR #158 统一视觉效果库与全量接入

- 用户批准建立覆盖 `AnimationPlayer / AnimationTree`、Tween / Curve、序列帧、`GPUParticles2D / CPUParticles2D`、CanvasItem Shader / 共享材质、精选程序几何复合模板及相机 / 音频 / 屏幕叠层的统一视觉表现层；明确排除离线 3D、AI / DCC 预渲染和节点式 VFX Graph。
- 新增 `visual_effects.json` 与 `presentation_profiles.json`。稳定 effect/profile id 由 DataLoader 校验；domain、kind、space、lifecycle、cue、anchor、quality、motion policy 和新增 pool / setting / RNG id 进入词表与生成契约。角色、敌人、武器、技能和机关以 `presentation_profile_id` 绑定，不在业务脚本手抄效果 id。
- 新增只读解析 autoload `VisualEffects`、run-scoped `VfxHost`、`GameplayFeedbackController`、`ActorPresentationController`、标准 `VfxAnchors`、请求 / 句柄 /引用资源、共享 Shader、Curve / target preset 与组合 `PackedScene`。独立表现随机使用 `RNG.vfx`；效果场景只负责配置、播放、取消、完成和池重置，不调用玩法系统。
- 现有真实伤害命中火花、三类飘字、玩家 / 敌人受击闪色、敌人 0.18 秒退场与脱离实体池的死亡余韵、技能成功 / 失败、Overdrive started / refreshed / expired / restored、状态 applied / restored / expired、枪口、弹体 RibbonTrail、拾取生成 / 吸附 / 收集和机关反馈已接入 profile/cue。击杀统计、掉落和移出活跃组仍即时发生，玩家有效受伤才震屏，命中停顿只保留接口且不驱动 `GameClock`。
- `UIManager` 升级为 `ENTERING → ACTIVE → EXITING → REMOVED`，保持 push/pop/replace 并新增 `pop_expected`、immediate clear、生命周期信号、导航去重、退出期暂停保持和下层焦点恢复；原生 ConfirmationDialog 被栈管理 `ConfirmationModal` 取代。Loading 等待 `ui_removed`，失败 / 硬切走 immediate。
- 新增 `UIEffectPlayer`、`UIPanelTransition`、`UIButtonFeedback`、`UISelectionFeedback`、`UIFocusIndicator`、`UIToast`、`UIValueFeedback`、`UIScreenAccent`、`UIEffectSequence`；Container 内只动画 `MotionRoot`，HUD 使用本地播放器而不进入 UIManager 栈。
- 新增设置 `accessibility.reduced_motion`、`accessibility.screen_flashes`、`video.vfx_quality`。Reduced motion 抑制可选运动效果，保留 gameplay boundary；飘字改为不位移的 100 ms 淡出，角色目标动画限制为不超过 100 ms，不暗改 `gameplay.screen_shake`；低质量只移除装饰层，不隐藏攻击边界、阵营或状态读法。
- 新增 editor-only `VFX Library` 中央主界面、预览场景 / 对象 / 空间、播放控制、三检查点、质量 / 动态 / 尺度 / 批量实例检查、Inspector 选择、Apply to Selected、模板 / profile 变体向导；release preset 排除 addon，正式资源禁止引用 editor-only 或 `output/test_lab`。
- 新增 `vfx-smoke` 与 `ui-manager-smoke`，并扩展 actor、runtime、settings、loading、schema、replay input 等回归。资源烘焙、DataLoader / Python 双校验、lint 与 lint 自测、UI / actor / runtime / save / loading / module-world smokes、headless editor/boot、实际 Windows PCK 导出和四条黄金回放均通过；黄金 gameplay summary 未变化，仅 catalog/profile 数据指纹按预期更新且未重录基线。事实型 code review 已完成并修复 target handle 取消、catalog pool 预热、长死亡余韵占池、RibbonTrail 重置、Curve 节奏、Reduced motion、Overdrive 生命周期与拾取吸附接线；按 ADR #143 未运行 startup/performance probe 或 Profiler。
