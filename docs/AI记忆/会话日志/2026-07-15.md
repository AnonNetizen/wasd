## Steamworks Slime Lab 单进程同屏合作

- 用户要求停止把本地联机建立在多进程 ENet Host/Join 上；新增 ADR #135，玩家可见本地多人改为单进程同屏，Steam 继续保持一台设备一名玩家，ENet 只保留内部协议自动回归。
- 新增 `SteamLabLocalInputRouter`：P1 固定键鼠，检测到的独立手柄按最低空槽分配 P2–P4；手柄使用左摇杆 / D-pad 移动、右摇杆瞄准、RT 开火、X 道具、A 合体、Y 表情、Start 暂停，支持 0.25 deadzone、最后瞄准保留、测试设备覆盖、阵容锁定、第五手柄忽略提示和最低缺失槽重绑。
- 主场景新增单人 / 本地同屏 / Steam Host / Steam Client 模式边界；同屏直接驱动权威战斗，不经过 RPC。移动、瞄准、开火 held / cooldown、主动道具、合体与表情意图改为逐玩家状态；P1 沿用保存外观，P2–P4 使用固定名和不同预设颜色。
- 玩家 UI 的地址、端口与 Join Local 已移除，准备房间显示 P1 键鼠和已检测手柄，至少一个手柄才可开始；大厅允许加入 / 移除，开战后锁阵容。掉线会冻结全场并保留角色状态，手柄补齐后转普通暂停，由玩家手动继续。
- 同屏 HUD 新增最多四张紧凑玩家卡；强化选择按存活槽位顺序逐人进行且无超时；BuffPanel、PausePanel 和 ExpressionWheel 增加当前玩家 / 设备控制接口，同一时刻只有一个表情轮但不阻断其他玩家。
