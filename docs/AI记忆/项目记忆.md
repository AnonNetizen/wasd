## 1. 项目快照

- **名称**：wasd
- **类型**：俯视角 Roguelike 弹幕生存（《以撒的结合》+《吸血鬼幸存者》风格）
- **引擎**：Godot 4.6.3 + GDScript（强制类型化）
- **当前状态**：**文档/框架设计阶段**，无代码
- **设计文档版本**：v1.5
- **仓库结构**：`docs/`（文档）、`client/`（Godot 客户端，占位）、`server/`（暂占位）
- **远端**：`https://github.com/AnonNetizen/wasd.git`
- **License**：MIT

## 2. 核心设计支柱（一眼回忆）

| 支柱 | 一句话 |
|------|-------|
| 数据驱动 | 所有可调数值进 `client/data/*.json`，零代码调参 + 热重载 |
| 遗物 = 数据 | `modifiers`（数值）+ `behaviors`（行为）描述，**严禁为每个遗物写硬编码分支** |
| **完整 autoload 矩阵** | 3 协作（`Localization`/`Settings`/`Analytics`）+ 2 确定性（`RNG`/`GameClock`）+ 1 回放（`Replay`）+ 3 协调（`GameState`/`UIManager`/`PoolManager`）+ 2 资源（`SaveManager`/`AudioManager`）+ AI 协作工程（`docs/AI协作/`） |
| 玩家恒居中 | 相机挂玩家下，**关 limit 与 drag margin** |
| 词表契约 | 所有约定字符串走 `docs/词表与契约.md` 白名单，**禁止裸字符串/编造 id** |
| AI 友好工程 | 项目索引、词表、数据校验、黄金样例、ADR、模板齐备；**任务模板 + 上下文预算 + 角色分工 + 实时验证回路** |
| 暂停 | `get_tree().paused` + 菜单节点 `process_mode=ALWAYS`，键走 InputMap action `pause` |
| 确定性 | 随机走 `RNG`、时间走 `GameClock`，禁止裸 `randi()` / `Time.get_ticks_msec()`（回放回归的硬约束） |

## 9. 更新约定（重要）

**触发条件（满足任一即更新）**：
- 任何 ADR 新增 / 修改
- 任何编码规则新增 / 修改
- 任何"待决策项"被采纳或新增
- 任何重要文件结构调整
- 任何工具链变更（CI、git 配置、新模板）
- 重要里程碑达成

**更新动作**（AI 主动执行，无需用户提醒）：
1. 修订本文件第 1~5 节（项目快照 / 决策 / 待决策 / 工具链）以反映现状
2. 在第 6 节追加当日条目（**一行表格条目 + 日志链接**，不展开细节）
3. 在 `会话日志/YYYY-MM-DD.md` 写当日详细摘要（输入/产出/决策）
4. 修订第 7 节"下一步候选"

**节制原则**：
- 只记**结论 + 关键参数 + 文件指针**，不复述完整对话
- 单次会话总更新量目标 < 1KB
- 历史细节归入会话日志，主索引保持精简（< 200 行最佳）

**自动瘦身（无需用户提醒）**：
- 每次更新本文件前先检查总行数；**临近 200 行立即瘦身**——不等用户问。
- 第 6 节单日条目超过 1 行 → 折叠为「日期 + 一句话摘要 + 日志链接」格式。
- 7 天前的条目 → 压缩为「YYYY-MM-W（周）：一句话」周聚合。
- 30 天前的条目 → 整体折叠为「月份 + 该月主要里程碑」月聚合。
- 决策清单（第 3 节）只保留编号 + 一句话；详情永远去 `决策记录.md` 查。
- 同一位置的累积修订 → **覆盖**而非追加（如 ADR 数量、CICD 项数）。
- 触发瘦身时本身**不算重大变更**，无需新加 ADR；仅在会话日志记一笔。

**不触发条件**（避免噪声）：
- 单纯的代码风格修复 / typo
- 不涉及决策的纯实现工作（在会话日志记一笔即可）
