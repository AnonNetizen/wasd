# 术语表

> **AI 修改说明**：修改本文档前先读 `docs/AI协作/文档维护指南.md`。
> 本文档是项目术语、别名和中英文检索词的权威索引；新增系统、玩法名、工具名、缩写或英文别名时，必须同步 `docs/_kb_index.json` 的相关关键词，必要时同步 `docs/词表与契约.md`。

---

## 1. 用法

- AI 搜索前先用本表扩展关键词，避免只搜中文或只搜英文导致漏结果。
- 本文档不登记约定字符串白名单；正式 id 仍以 `docs/词表与契约.md` 为准。
- 本文档允许记录同义词、缩写、口语说法和英文 API 语境。

## 6. 常见误用

| 误用 | 正确说法 | 原因 |
|------|----------|------|
| 道具泛指所有内容 | 区分主动道具、消耗品、遗物 | 数据结构和扩展点不同 |
| 手柄轴直接写代码里 | InputMap action | 设备差异必须在绑定层 |
| 自动生成文档 | 自动创建/维护详细模块文档 | 不能用简短摘要替代维护文档 |
| MVP 代码经验直接进正式项目 | 先复盘，再 ADR / GDD / 词表同步 | MVP 是隔离实验区 |
| 词表当术语表 | 词表登记正式 id，术语表登记别名 | 权威范围不同 |
